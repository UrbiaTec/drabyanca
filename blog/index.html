Como Editar o Blog

Para administradores: Este blog usa arquivos Markdown (.md) para facilitar a edição:

  • Posts: Adicione novos posts na pasta posts/ seguindo o padrão de nomenclatura YYYY-MM-DD-titulo-do-post.md
  • Categorias: Crie novas categorias na pasta categorias/ com o nome nome-da-categoria.md
  • Atualizar esta página: Edite o arquivo index.html para adicionar links para novos posts e categorias
  • Estilos: O CSS está no arquivo blog-style.css

Após adicionar novos conteúdos, lembre-se de atualizar os links nesta página para que apareçam na lista de posts e categorias.
